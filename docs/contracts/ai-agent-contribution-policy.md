# AI and Agent Contribution Policy

## Policy

AI-assisted and agent-authored contributions are allowed with mandatory disclosure and human accountability.

## Required disclosure in PR

1. Tool/model used
2. Contribution mode (`ai-assisted` or `agent-authored`)
3. Which sections were generated
4. Human sponsor/reviewer

## Non-negotiable controls

- Human review before merge
- Contract and fixture checks pass
- Playwright evidence attached for user-facing impact
- Security scan/SBOM gates pass

## Prohibited

- Merging AI output without review
- Committing secrets or local absolute paths
- Presenting fabricated references as verified evidence
- Importing unlicensed code without compliance review
