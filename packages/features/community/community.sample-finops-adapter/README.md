# community.sample-finops-adapter

Reference starter module for community contribution workflow.

## Purpose

Demonstrate minimum required structure for a community module:

- contract-aware `feature.json`
- deterministic normalization function
- test stub
- fixture parity bundle

## Governance

- trust tier: `tier-1`
- contribution type: `ai-assisted` (placeholder for demonstration)
- sponsor required for merge

## Paths

- manifest: `feature.json`
- implementation: `src/index.ts`
- tests: `tests/community.sample-finops-adapter.test.ts`
- fixtures: `tests/contracts/fixtures/community.sample-finops-adapter/0.1.0/`
